# 🐾 PawCare: Municipal Pet Management System

**PawCare** is an advanced, data-driven Municipal Pet Management System designed to streamline pet registration, vaccination tracking, and clinical appointments for local government units.

---

## 🚀 Key Features

### 📋 Smart Pet Records
- **Digital Registration**: Comprehensive onboarding for owners and pets.
- **Archive Center**: Advanced record preservation for 'INACTIVE' or deceased pets to maintain biological data integrity.
- **Dynamic Identification**: Unique Pet ID generation and digital profiling.

### 📅 Intelligent Appointment System
- **Real-time Availability**: Interactive calendar with automated slot occupancy calculation (Green for Available, Red for Full).
- **Address Integration**: Captures appointment-specific location data to track local health trends and disease outbreaks by barangay.

### 💉 Medical & Inventory Sync
- **Atomic Transactions**: Real-time coupling between medical history and vaccine inventory.
- **Auto-due Logic**: Automatic calculation of next vaccination booster dates.
- **Low-Stock Alerts**: Proactive notifications for clinic staff when vaccine supplies are low.

### 🔍 QR Code Verification
- **Rapid Field Identification**: Scannable QR codes that resolve to public verify pages.
- **Optimized Admin Workflow**: QR scans directly trigger the 'View Profile' modal for instant verification without manual searching.

### 🛡️ Enterprise-Grade Security
- **3-Tier RBAC**: Distinct roles for Admin, Staff, and Pet Owners.
- **Middleware Protection**: Server-side role validation on every request.
- **Full Audit Trail**: Comprehensive logging of every system action, status change, and login attempt.
