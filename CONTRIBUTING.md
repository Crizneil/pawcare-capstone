# Contributing to PawCare

Thank you for your interest in contributing to the PawCare Capstone project!

## How to Contribute
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/amazing-feature`).
3. Commit your changes.
4. Push to the branch.
5. Open a Pull Request.

## Coding Standards
- Follow Laravel coding standards.
- Ensure all new features are documented.
- Respect the mission of providing better health tracking for pets.

🐾 Together we make pet care better!
